This page fetches the BBC Weather RSS 3day forecast feed for a given location ID, which fortunately also works for the first part of a UK postcode. If it's not returning data, or you want to use a more precise location, first visit https://bbc.co.uk/weather, search for your location, then copy the 7-digit number from the url (the last part). For example, searching for "Manchester" and choosing the first result, "Manchester, Manchester", results in https://www.bbc.co.uk/weather/2643123. Therefore using 2643123 in the form below will give you a forecast for Manchester, UK.
